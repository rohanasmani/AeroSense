# AeroSense
AeroSense - An ESP32-based LiDAR object detection project for real-time obstacle detection and distance measurement using embedded systems and sensor interfacing.

## Overview

This project uses the ESP32 for sensor interfacing, data processing, and communication. The LiDAR sensor measures the distance of nearby objects, and the ESP32 processes this data for monitoring and detection applications.
---
## Features

* MPU6050 Stabilization
* VL53L0X Obstacle Detection
* Intelligent Anti-Crash
* Smooth Motor Control
* Arm / Disarm
* Flight Ready Logic
* Stable Lidar Initialization
* Serial Monitoring

---

## Applications

* Obstacle avoidance robot
* Indoor mapping
* Smart navigation
* Autonomous vehicle sensing
